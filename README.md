# night-saver-web
야간 주행을 하던 중 전조등이 켜져있지 않은 차량으로 인해 차선 변경이나 주행중 놀라는 경험을 많이 하였습니다. 
이로 인한 불편함을 조금이라도 덜 고자 야간 주행 혹은 어두운 환경(터널 등)에서 차량의 전조등이 켜져 있을 수 있다면 어떨까 싶어 어플리케이션을 제작하였습니다.
> 데스크톱 환경보다는 모바일 친화적으로 제작하였습니다.
